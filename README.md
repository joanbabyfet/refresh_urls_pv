## About
一个网站刷量脚本，根据 urls.txt 名单来进行

## Feature

* 循环访问 urls.txt 名单里网址，达到刷量效果
* 随机 user-agent (ua.txt 可自行添加修改)
* 随机浏览器窗口大小 (windowsize.txt 可自行添加修改)
* 不同代理ip (proxy.txt 可自行添加修改)

## Requires
Python 3.11.0  
selenium 4.8.0  
webdriver_manager 3.8.5   
fake_useragent 1.1.1  

## Usage
```
python main.py
```
